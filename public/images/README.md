# Image folder – Print Studio

Place your images here so the site uses them.

## Service cards (Home page) – 4 images

Put these files directly in `public/images/`:

| File name           | Used for      |
|---------------------|---------------|
| visiting-card.jpg   | Visiting Card |
| wedding-card.jpg   | Wedding Card  |
| shop-banner.jpg     | Shop Banner   |
| flex-printing.jpg  | Flex Printing |

## Gallery (Category pages) – 10 images per category

Create subfolders and name images `1.jpg` to `10.jpg`:

- `public/images/visiting/` → 1.jpg, 2.jpg, … 10.jpg  
- `public/images/wedding/`  → 1.jpg … 10.jpg  
- `public/images/banner/`   → 1.jpg … 10.jpg  
- `public/images/flex/`     → 1.jpg … 10.jpg  

If you use **.png** instead of .jpg, change `EXT` in `src/data/designs.js` to `'.png'`.
